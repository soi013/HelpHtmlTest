# 🍪HOW TO USE🍪

丸いところをクリックすると<kbd>Coockie Count</kbd>が1つ増えます。

## UI説明



| 項目           | 見た目   | 説明                       |
| -------------- | -------- | -------------------------- |
| Coockie Count  | 文字     | 現在の焼けたクッキー数     |
| Coockie Button | クッキー | 押すとクッキーが焼き上がる |
